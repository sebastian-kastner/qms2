Class App_Process_Broker

Description
Description | Vars (details) | Methods (details)

Dient zum Zusammenstellen von Prozessinformationen (vor allem Prozessattribute) zu einem einzigen Prozess.

Die Attribute können von verschiedenen Typen sein. Die Attribute können entweder reinen Text enthalten oder eine Referenz auf eine Methode, deren Rückgabewert dann als Attributwert gespeichert wird
