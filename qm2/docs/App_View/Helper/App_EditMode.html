Class App_EditMode

Description
Description | Vars (details) | Methods (details)

Dient mit statischen Methoden zum Setzen und Auslesen des Bearbeitungsmodus.

Der Bearbeitungsmodus wird als BOOLEAN gespeichert. Befindet der User sich aktuell im Bearbeitungsmodus ist der der $_editMode folglich TRUE, wenn nicht folglich FALSE
